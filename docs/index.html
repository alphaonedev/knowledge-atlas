Distilled knowledge from any expert, for humans and AI.

A self-hosted pipeline that turns any subject-matter expert's video corpus into a queryable knowledge base. Localhost-first. MCP-native. The deliverable is paraphrased, attributable knowledge — not raw transcripts and not statistics.

Install Read the architecture

What it produces

For every video in an indexed source, an LLM produces 4–12 knowledge cards — each a standalone, paraphrased nugget classified by kind, with provenance back to a specific timecoded YouTube source.

principle tactic warning framework mental model phrase quote

Three ways to consume

For humans

Web dashboard

A polished localhost SPA. Browse by topic, kind, or source. ⌘K to search. Click into any card for the cited video.

For agents

HTTP/JSON API

Stable endpoints for autonomous agents — /ai/manifest, /ai/teach, /ai/cross/concept/<term>. Discovery built in.

For Claude / ChatGPT / Cursor

MCP server

Drop-in tool surface for Claude Desktop, Claude Code, ChatGPT Desktop, Cursor, Cline, and any MCP client. 14 tools, stdio JSON-RPC. See the tool reference →

MCP tools reference

14 tools across four categories — browse the corpus, query it, compare experts side-by-side, and manage sources. Browse / query / compare tools read SQLite directly and work even when the dashboard is offline; manage tools mutate state and call back into the Flask service.

System overview

All four pipeline stages feed one SQLite store. Four parallel surfaces read from it — three over HTTP, one over MCP stdio.

Why it's valuable

Knowledge, not metrics

No sentiment scores, no readability indices, no word counts. The deliverable is paraphrased, comprehensible nuggets a human can absorb in seconds.

Cross-source by design

Every card carries a source_id. Index 2+ experts and /ai/cross/* surfaces consensus, conflict, and coverage automatically.

One schema, any LLM

The card contract is fixed. Swap xAI Grok for Anthropic Claude or any other provider by writing one function — the atlas doesn't care who extracted.

Localhost trust model

Binds to 127.0.0.1 by default. No cloud, no telemetry, no remote auth. API keys live in a chmod-600 .env. Your atlas stays on your machine.

Always cite the source

Every card includes video_url. The AI you point at this surface can always hand the human a verifiable timecoded link.

Self-serve onboarding

Click Index expert in the dashboard, paste a YouTube URL, watch the pipeline run live. No CLI required after install.

Try a teaching prompt

Once Claude Desktop or Claude Code is wired in via MCP, ask:

Teach me about cross-examination using the knowledge-atlas tools.

What do my indexed experts say about negotiating with a narcissist?
Cite source videos.

Show me cross-source coverage and tell me which topics are deepest.

Ready to install?

Five minutes to first knowledge card. Two minutes to wire Claude Desktop in via MCP.
